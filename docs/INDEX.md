# docs/ ‚Äî index & currency map

Every doc under `docs/`, its one-line purpose, and a **currency tag** so a new
reader knows what to trust. Tags:

- **CURRENT** ‚Äî verified against code; load-bearing claims hold.
- **DECISION RECORD** ‚Äî a frozen ADR in `decisions/`: the decision stands, the
  point-in-time system descriptions are not maintained for drift.
- **STALE ‚Äî see note** ‚Äî contains at least one claim the code contradicts; note says what.

Currency is verified by grepping each doc's load-bearing claims against the code, not
by trusting the doc's own "Status:" line. When you change behavior that a doc
describes, fix the doc **and** update its tag here in the same PR.

## Layout

```
docs/
‚îú‚îÄ‚îÄ architecture.md    the system doc ‚Äî mental model + the locked contract
‚îú‚îÄ‚îÄ flows/             temporal traces (one task; dispatch ‚Üí PR)
‚îú‚îÄ‚îÄ reference/         look-up tables (env vars)
‚îú‚îÄ‚îÄ runbooks/          operational procedures (shakedown, VPS deploy)
‚îú‚îÄ‚îÄ decisions/         frozen ADRs ‚Äî rationale, not current state
‚îî‚îÄ‚îÄ proposals/         pre-ADR staging ‚Äî direction drafts (DRAFT ‚Üí LOCKED ‚Üí ADR)
```
